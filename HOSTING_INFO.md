# 호스팅 관리자 페이지 설명 (Hostinger)

업로드해주신 사진은 호스팅 서버를 관리하는 **Hostinger 제어판** 화면입니다.

## 1. 파일 관리자 (File Manager)
-   **용도**: 홈페이지 파일(HTML, PHP, 이미지 등)을 서버에 업로드하는 곳입니다.
-   **사용법**:
    1.  두 번째 사진에서 **왼쪽 아이콘 (보라색 문서/sejongcook.co.kr)**을 클릭하세요.
    2.  `public_html` 폴더를 찾아 들어갑니다.
    3.  그곳에 작업하신 빌드 파일(`api.php`, `index.html`, `assets` 폴더 등)을 덮어쓰거나 업로드하시면 됩니다.

## 2. 데이터베이스 (MySQL Databases)
-   **용도**: 게시판, 공지사항, 질문글 등을 저장하는 공간입니다.
-   **사용법**:
    1.  세 번째 사진 화면에서 새로운 데이터베이스를 만듭니다.
    2.  **MySQL 데이터베이스 이름**: 예) `sejong_db`
    3.  **MySQL 사용자 이름**: 예) `admin`
    4.  **비밀번호**: (잊어버리지 않게 메모 필수!)
    5.  **생성** 버튼 클릭.

## 3. 연결 (중요)
데이터베이스를 만든 후, **DB 이름, 사용자명, 비밀번호**를 `api.php` 또는 `db_connect.php` 파일 설정에 입력해줘야 홈페이지에서 글쓰기가 가능해집니다.

## 4. 추가 질문: 오른쪽 폴더 아이콘은 무엇인가요?
사용자분이 질문하신 **"Premium Web Hosting의 모든 파일에 접근합니다" (오른쪽 폴더 아이콘)** 버튼에 대한 설명입니다.

*   **기능**: 호스팅 계정 내의 **모든 웹사이트 파일**을 관리하는 곳입니다. (여러 사이트를 운영할 때 유용함)
*   **사용법**:
    1.  이 버튼을 클릭하면 최상위 폴더가 나옵니다.
    2.  `domains` 폴더를 더블 클릭합니다.
    3.  `sejongcook.co.kr` 폴더를 더블 클릭합니다.
    4.  `public_html` 폴더를 더블 클릭합니다.
    5.  여기에 파일을 업로드하면 **왼쪽 버튼(사이트 전용)**으로 들어갔을 때와 똑같이 적용됩니다.

**팁**: 경로를 찾아들어가기 복잡하다면, **왼쪽의 보라색 문서 아이콘(사이트 전용)**을 쓰시는 것이 더 간편합니다.

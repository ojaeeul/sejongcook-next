# 세종요리제과학원 서버 통합 실행 가이드

컴퓨터를 재부팅한 후, 다음 단계를 통해 모든 시스템을 한 번에 실행할 수 있습니다.

## 1. 실행 방법
1. 바탕화면이나 프로젝트 폴더에서 **`start_servers.command`** 파일을 찾습니다.
2. 해당 파일을 **더블 클릭**합니다.
    - 보안 경고가 뜰 경우: `우클릭 -> 열기`를 선택해 주세요.

## 2. 자동 수행 항목
- **데이터 동기화**: 가장 최신 수강생 명단(9명)을 자동으로 적용합니다.
- **출결 서버 (8000)**: 출결 관리 및 납부대장 시스템을 시작합니다.
- **웹 서버 (3000)**: 메인 홈페이지 시스템을 시작합니다.
- **브라우저 자동 열기**: 납부대장, 출결관리, 홈페이지 화면이 자동으로 열립니다.

## 3. 주의 사항
- 터미널 창이 열리면서 서버가 실행됩니다. 작업이 끝날 때까지 해당 **터미널 창을 끄지 말아주세요.** (최소화해두시면 편리합니다.)
- 종료하시려면 터미널 창에서 `Ctrl + C`를 누르거나 창을 닫으시면 됩니다.

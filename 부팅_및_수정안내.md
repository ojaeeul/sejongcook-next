# 세종요리 시스템 부팅 및 수정사항 안내

## 1. 서버 시작 방법
루트 폴더에 생성된 **`시스템_시작.bat`** 파일을 더블 클릭하면 다음 서버들이 자동으로 실행됩니다:
- **메인 웹사이트**: [http://localhost:3000](http://localhost:3000)
- **출석 데이터 백엔드**: Port 8000

## 2. 주요 수정 내용 (저장 완료)
요청하신 모든 수정 사항이 소스 코드에 영구적으로 반영되었습니다:

### 게시물 가시성 문제 해결
- **미들웨어(Middleware) 수정**: `/api` 경로가 관리자 인증에 막혀 게시물이 안 보이던 현상을 해결했습니다. 이제 홈페이지의 모든 소식이 정상적으로 로드됩니다.
- **"미리보기" 섹션 추가**: "새로운 소식" 섹션에 4번째 컬럼으로 **'미리보기'(미리보기)**를 추가했습니다. 이 섹션은 관리자 페이지의 '팝업' 데이터를 실시간으로 가져옵니다.
- **데이터 정렬 및 안전 로직**: 공지사항, 질문&답변, 취업정보 등이 항상 최신순으로 정렬되도록 수정하였으며, 데이터가 없을 때 무한 로딩되는 현상을 방지했습니다.

### 시스템 호환성
- **휴대용 노드(Portable Node) 설정**: 폴더 내의 `.node`를 사용하도록 배치 파일(`시스템_시작.bat`)을 구성했습니다. 이제 다른 PC로 폴더를 옮겨도 별도의 노드 설치 없이 실행 가능합니다 (Python은 설치 필요).

## 3. 주의 사항
- 서버 실행 중에는 열린 검은색 창(콘솔)을 닫지 마세요. 창을 닫으면 해당 서버가 종료됩니다.
- 만약 `node_modules` 관련 오류가 발생하면, `시스템_시작.bat` 실행 전 터미널에서 `npm install`을 한 번 실행해 주세요.
